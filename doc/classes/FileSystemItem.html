timestamp

Integer

Defined in js/agora-interface.js:122

The timestamp is an integer representation of when the file was last modified. It is formatted as follows:

YYYYMMDDHHMMSS

Where:

-YYYY = Year
-MM = Month
-DD = Day
-HH = Hour
-MM = Minute
-SS = Second

Default: 0
